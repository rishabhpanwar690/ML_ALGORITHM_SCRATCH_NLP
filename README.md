# ML_ALGORITHM_SCRATCH_NLP
Applicant Screening Predictor: ML Algorithms from Scratch
Overview
This project implements a custom ensemble of 12 Machine Learning algorithms built entirely from scratch using NumPy. It uses Natural Language Processing (TF-IDF) to analyze applicant transcripts, resumes, and job descriptions to predict whether a candidate should be selected or rejected.

Key Features
12 Algorithms from Scratch: Includes Linear/Ridge Regression, Logistic Regression, Decision Trees, Random Forests, KNN, Naive Bayes, Perceptron, and Linear SVM—implemented without sklearn's model classes.

NLP Pipeline: Uses TfidfVectorizer to convert text features into a numerical matrix.

Custom Ensemble Method: A weighted average voting system based on the validation accuracy of the individual scratch models.
